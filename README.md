# Enterprise Real-Time Chat Platform

## Overview

Enterprise Real-Time Chat Platform is a professional chat application built using Flask, Socket.IO, MongoDB, JWT Authentication, and Bootstrap.

The platform enables secure communication through real-time messaging, chat rooms, typing indicators, online user tracking, and persistent message storage.

---

## Features

- User Registration
- Secure Login Authentication
- JWT Token Authentication
- Password Hashing using Bcrypt
- Real-Time Messaging
- Multiple Chat Rooms
- Online User Tracking
- Typing Indicators
- Dark Mode Support
- MongoDB Message Storage
- Responsive Bootstrap Interface
